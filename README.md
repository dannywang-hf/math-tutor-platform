# 小学数学一对一辅导平台

> 专为六年级学生设计的数学学习平台，提供个性化辅导、练习考试、AI答疑、PPT演示+人声讲解等功能。

## 功能特点

### 📚 课程学习
- 按章节组织六年级数学知识点
- 包含分数乘法、分数除法、比和比例、圆、百分数五大章节
- 每个知识点配有详细讲解和例题
- **📺 PPT视频讲解** - 每节课都有精心制作的幻灯片演示
- 学习进度自动保存

### 🎬 PPT演示 + 人声讲解
- **每节课配套视频讲解**：精美的幻灯片演示，配合人声朗读
- **自动播放模式**：可以自动翻页播放讲解
- **手动控制**：可自由切换幻灯片、暂停播放
- **TTS语音合成**：使用浏览器原生语音，播放讲解内容
- **错题视频讲解**：每道错题都有PPT形式的详细解析

### 🔄 复习巩固
- 自动识别薄弱知识点
- 根据错题智能推荐复习内容
- 薄弱点专项练习

### ✏️ 练习考试
- **快速练习**：随机10道题快速检测
- **章节练习**：针对特定章节练习
- **薄弱点专练**：针对薄弱环节强化训练
- **模拟考试**：20道题完整测试
- **错题视频解析**：每道错题都有PPT+人声讲解

### 📊 学情分析
- 可视化学习进度图表
- 知识点掌握雷达图
- 学习趋势分析
- 个性化学习建议

### 👨‍🏫 AI辅导
- 24小时在线答疑
- 支持常见数学问题快速回复
- 学习小贴士每日更新

## 使用方法

### 基础使用

1. **打开网页**：用浏览器打开 `index.html` 文件
2. **注册/登录**：首次使用需要注册账号
3. **开始学习**：
   - 点击"课程学习"学习新知识
   - 点击"练习考试"做练习题
   - 点击"AI辅导"随时提问

### 观看视频讲解

#### 课程视频
1. 进入"课程学习"页面
2. 选择章节和课程
3. 点击"📺 视频讲解"按钮
4. 在PPT演示窗口中：
   - 点击 **▶️** 按钮启动自动播放
   - 使用 **◀ ▶** 按钮手动翻页
   - 点击 **🔊** 开关可开启/关闭语音
   - 点击 **×** 关闭窗口

#### 错题视频
1. 完成练习后查看错题
2. 点击"📺 视频讲解"按钮
3. 观看该题的详细PPT解析

## 技术说明

- **纯前端实现**：数据存储在浏览器本地
- **TTS语音**：使用 Web Speech API，无需额外插件
- **无服务器依赖**：直接打开 HTML 文件即可使用
- **支持浏览器**：Chrome、Edge、Firefox、Safari 等现代浏览器

## 常见问题

### Q: 为什么没有声音？
A: 请确保浏览器允许网页播放音频。首次使用可能需要点击页面让浏览器激活音频上下文。

### Q: 可以离线使用吗？
A: 可以！首次加载后，所有资源都会缓存到浏览器中，可以离线使用。

### Q: 数据会丢失吗？
A: 数据存储在浏览器本地，清除浏览器数据会丢失。建议定期导出学习记录。
